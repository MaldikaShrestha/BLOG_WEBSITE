##DJANGO BLOG WEB APPLICATION
A full featured web blog application that is built using using Python Django Framework where users can create, edit and manage their posts. This project demonstrates the core Django concepts like model, views, templates, database integration and authorization.

##FEATURES
- Authentication and registration (Existing users can loggin to their account using password and new users can register and create a new account)
- Create a profile and update profile
- Create, Edit and Delete Post(only by the author of the post)
- View the posts created by other users
- Admin pannel for managing posts and the users
- Responsive design
- User can reset their password through email

  ##TTECHNOLOGY STACK
  - PYTHON 3.3
  - DJANGO
  - HTML,CSS,JAVASCRIPT
